Strings em JavaScript


O que é uma String?

São sequências de caracteres alfanuméricos (letras, números e/ou símbolos) delimitados por aspas simples ‘exemplo’ ou aspas duplas “exemplo”. 

                    
    var saudacao = "Olá!";                  
    var nome = "Henning Summer";

As strings podem conter qualquer combinação de caracteres, incluindo letras, números, espaços, símbolos e até mesmo caracteres especiais, como acentos.


Atenção aos Detalhes! 

                    
    var  exe1 = 350;
    var  exe2 = "350";

No exemplo acima, por mais que o valor de ambas as variáveis seja o mesmo, essas variáveis possuem tipos primitivos diferentes e não podem ser tratadas como iguais.

A primeira var é do tipo Number e a segunda var é do tipo String. Atenção as “aspas”.


Aspas dentro de Aspas

Em alguns momentos, a string poderá ser um texto que contém aspas. Nesses casos, é preciso combinar a utilização das aspas ‘simples’ com “aspas duplas” e vice-versa. 

                    
    var exe1 = 'Ela disse: "Adeus!"';
    var exe2 = "Ela disse: 'Adeus!'";



Em JavaScript as Strings são imutáveis

Uma vez que uma string é definida, ela não pode ser modificada. No entanto, existem formas de se manipular strings criando novas strings com base na original. Para isso são utilizadas operações ou métodos de manipulação de strings.



Manipulando Strings


Convertendo os outros tipos primitivos em tipo string

Lembrando que os tipos primitivos existentes em JavaScript são: Number, Boolean, Null, Undefined, Symbol, além é claro do tipo String. Sendo assim 6 ao todo.

É possível fazer a conversão de qualquer um desses outros tipos para o tipo string através do método toString(). 

                    
    var booleano = true
    console.log(typeof booleano) // Saída: boolean
                      
    var booleanoString = booleano.toString()
    console.log(typeof booleanoString) // Saída: string

Também é possível obter o mesmo resultado de uma forma diferente, criando um novo objeto usando new String(parâmetro), sendo o parâmetro o elemento que irá ser transformado em string. 

                    
    var booleano = true;
    var booleanoString = new String(booleano);
    console.log(booleanoString); // Saída: True


Concatenando strings

Na programação, a concatenação de strings significa criar uma nova string a partir da junção de duas ou mais strings existentes. 

                    
    var nome = "André";
    var sobrenome = "Silva";
    var nomeCompleto = "Meu nome completo é:" + nome + " " + sobrenome;
    console.log(nomeCompleto) // Saída: Meu nome Completo é: André Silva

Neste exemplo, uma nova string nomeCompleto foi criada a partir da concatenação das strings nome e sobrenome. A concatenação é feita através do operador de adição +.

Também é possível utilizar os operadores += para fazer a concatenação. 

                    
    var nome = "André";
    var saudacao = "Seja bem-vindo(a)";
    saudacao += nome
    console.log(saudacao) // Saída: Seja bem-vindo(a) André


Interpolando strings (template strings)

É uma alternativa mais prática para manipular strings sem a necessidade de fazer a concatenação. Usando as chamadas template strings ou templates literais, é possível ter uma maior flexibilidade no trabalho com strings, além de facilitar a escrita e leitura do código.

Para utilizar template strings, você deve delimitar a string com acento grave `exemplo` em vez de aspas ‘simples’ ou “duplas”. Dessa forma, é possível inserir expressões e variáveis diretamente dentro da string. 

                
    var idade = 16;
    var frase = `Meu nome não é Johnny e eu tenho ${idade} anos de idade`;
    console.log(frase) // Saída: Meu nome não é Johnny e eu tenho 16 anos de idade

Neste exemplo, envolvemos a frase com o acento grave ` ` em vez das aspas que normalmente usamos por padrão, e dentro da frase usamos a expressão ${idade}, sem a necessidade da concatenação.


Tamanho de uma string

O método length serve para informar o tamanho (número de caracteres) de uma string. 

                    
    var sring = "Hello, World!";
    var comprimento = string.length;
    console.log(comprimento) // Saída: 13

É importante observar que o length conta todos os caracteres presentes na string, incluindo espaços em branco e pontuação. Ele também conta os caracteres especiais e acentos de forma individual.


Acessando a posição de uma string

Através do método charAt() conseguimos acessar um caractere de uma string. Lembrando que, por baixo dos panos, strings são arrays de caracteres, e em cada posição temos o caractere que compõe a string. 

                        
        var alunoNota10 = "Germano";

        var comprimento = alunoNota10.length;
        console.log(comprimento) // Saída: 7

        console.log(alunoNota10.charAt(3)) // Saída: m
        console.log(alunoNota10[3]) // Saída: m

O nome Germano possui ao todo 7 caracteres, porém a contagem é iniciada pelo número 0. O primeiro caractere de uma string é acessado pelo índice 0, o segundo caractere pelo índice 1 e assim por diante.

A contagem dos caracteres iniciar de 0 ocorre por causa da forma como as strings são armazenadas e acessadas na memória.


Posição de um caractere dentro da string

A função indexOf() retorna a posição de um caractere dentro da string. Neste caso, basta informar qual caractere deseja saber a posição. 

                    
    var alunoNota10 = "Germano";

Convertendo uma string em letras maiúsculas e/ou minúsculas

toUpperCase() e toLowerCase() são duas funções usadas para converter uma string em letras maiúsculas (upper case) ou letras minúsculas (lower case). 

                    
    var palavra = "JavaScript";
    console.log(palavra.toUpperCase()) // Saída: JAVASCRIPT
    console.log(palavra.toLowerCase()) // Saída: javascript

Estes métodos fazem a manipulação da string como um todo.


Extraindo parte de uma string

A função substring, substr() permite que seja feita a extração de parte de uma string. 

                    
    var frase = "Mergulhando em tecnologia.";
    Console.log(frase.substr(0,11)) // Saída: Mergulhando
    
    Console.log(frase.substr(15)) // Saída: tecnologia.

    Console.log(frase.slice(0,11)) // Saída: Mergulhando

A função recebe como parâmetro o início e o fim da nova string a ser retirada da string principal. Na execução do código acima, temos como resultado a palavra Mergulhando (primeiro print).

Já o segundo print, foi especificado que a extração da string iniciaria do índice 15 em diante, o que resulta em: tecnologia. (também funciona no método slice()).

Um outro método similar ao substring é o método slice(), usado em arrays. Ele retornará parte de uma string, desde que sejam passados os índices de início e de fim. 

                    
    var frase = "Mergulhando em tecnologia.";
    Console.log(frase.slice(0,11)) // Saída: Mergulhando


Substituindo parte de uma string

Com a função replace() temos a possibilidade de substituir parte de uma string por outra. Essa função recebe como parâmetros duas informações: a string que você quer substituir e a string que será colocada no lugar. 

                    
    var nome = "Flavinha";
    var mensagem = "Feliz Aniversário userName! Parabéns pelos 15 anos.";
    console.log(mensagem.replace("userName", nome)) // Saída: Feliz Aniversário Flavinha! Parabéns pelos 15 anos.
